---
name: interactive-nr-report
description: Interactive recipe to pick a New Relic app and generate a PDF performance report.
tags: [newrelic, interactive, pdf]
---

## step-list-apps
tool: newrelic
operation: list_apps

## step-pick-app
tool: system
operation: ask
condition: ${step-list-apps.success} == true
question: |
  I found the following New Relic entities: ${step-list-apps.summary}.
  Please enter the GUID of the application you want to report on (you can find it in list-apps-context.md):

## step-fetch-errors
tool: newrelic
operation: query
scenario: errors
timeWindowMinutes: 7200
filters:
  entityGuid: ${step-pick-app.answer}

## step-process-data
tool: json
operation: extract
data: ${step-fetch-errors.body}
regex:
  error_trend: '"count":\s*(\d+)'

## step-gen-report
tool: pdf
operation: create
path: newrelic-interactive-report.pdf
content: |
  NEW RELIC PERFORMANCE REPORT
  ============================
  Target Application: ${step-pick-app.answer}
  Analysis Window: Last 5 days
  
  ERROR VOLUME ANALYSIS:
  The following trend shows error volume over the last hour.
  
  chart_data: ${step-process-data.error_trend}
  
  Report generated by ProdBuddy interactive workflow.
